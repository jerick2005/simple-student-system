# Resume Management System

A modern, responsive web application for managing professional resumes using HTML, CSS, and JavaScript.

## Features

- Add new resumes with personal information, profile image, skills, and work experience
- Upload and display profile images
- View all resumes as interactive cards
- Edit existing resumes
- Delete resumes
- Data persistence using browser's LocalStorage
- Modern, responsive design with animations
- Mobile-friendly interface
